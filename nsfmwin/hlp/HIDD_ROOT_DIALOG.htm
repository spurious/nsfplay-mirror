NSFMeasure 基本画面

このダイアログボックスに演奏時間を検出したいNSFファイルまたはNSFファイルを含むフォルダをドラッグ＆ドロップしてください。リストボックスにNSFファイルが追加されます。

Start ボタン

演奏時間の検出を開始します。リストボックスに表示されているNSFファイルが上から順番に読み込まれ、処理されます。検出結果はNSFファイルと同じフォルダのtaginfo.tagファイルに書き込まれます。検出中はこのボタンの名称がStopに変わります。このボタンを再度押すことで、検出を中断することができます。

Delete ボタン

リストボックス内で選択されている項目を削除します。

Up ボタン

リストボックス内で選択されている項目を、ひとつ上に移動させます。

Down ボタン

リストボックス内で選択されている項目を、ひとつ下に移動させます。

Option ボタン

オプション画面を開きます。

Help ボタン

ヘルプ画面を開きます。
